# Marketing Campaign Effectiveness

## Description
Analyze marketing campaigns to measure ROI, CAC, Conversion Rate, and channel performance.

## Tools
Python, Pandas, Matplotlib, (Optional: Tableau)

## Data
Synthetic marketing campaign dataset (CSV).

## Instructions
1. Open `notebooks/Marketing_Campaign_Analysis.ipynb`.
2. Run all cells to see KPI summary and charts.
3. KPIs are saved in `data/channel_kpis.csv`.

## Optional Dashboard
Screenshots saved in `dashboards/`.
